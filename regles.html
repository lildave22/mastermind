Règles du MasterMind

Bienvenue dans le jeu du Mastermind ! Voici les règles du jeu : un code a été généré aléatoirement, et c'est à vous de retrouver ce code dans le temps imparti qui est de 30 secondes. Entrez chaque chiffre qui composent le code, si le chiffre est le bon à la bonne position, un '1' s'affiche; si le chiffre est bon mais à la mauvaise place, un '.' s'affiche; sinon, une 'X' s'affiche. Bonne chance !
